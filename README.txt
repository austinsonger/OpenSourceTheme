To install this theme in /r/your_subreddit:

	1.	Be logged in as a moderator of /r/your_subreddit.
	2.	Go to www.reddit.com/r/your_subreddit/about/stylesheet/
	3.	Paste the .css file contents into the css text area and save.
	4.	Go to http://www.reddit.com/r/AskCoffee/about/edit/ and at the bottom of the page, click "Choose file" and upload the image file. Save.

	You're done!
